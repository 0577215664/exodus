netpacket
=============

.. contents::

net packet decode and encode.


支持
----------

*OSI Model Layer 1*

*   Ethernet II Packet


*OSI Model Layer 2*

*   Regular Ethernet II frames
*   ARP
*   [TODO] NDP


*OSI Model Layer 3*

*   IPv4
*   [TODO] IPv6
*   ICMPv4
*   [TODO] ICMPv6
*   [TODO] IGMP


*OSI Model Layer 4*

*   TCP
*   UDP

*OSI Model Layer 5*

*   [TODO] DHCP
*   [TODO] DNS
