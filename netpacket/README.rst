netpacket

.. contents::

net packet decode and encode.


支持
----------

*OSI Model Layer 1*



*OSI Model Layer 2*

*   Regular Ethernet II frames

*OSI Model Layer 3*

*   IP(V4/V6)


*OSI Model Layer 4*

*   TCP/UDP
